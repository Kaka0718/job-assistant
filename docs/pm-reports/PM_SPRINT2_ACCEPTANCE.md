## 4. 核心功能验证

### 4.1 AI 打招呼生成流程

```
用户操作 → 粘贴 JD → 选择岗位档案 → 点击"生成打招呼"
    ↓
JD 校验 → 档案校验 → API Key 校验 → 个人档案校验
    ↓
AI API 调用（OpenAI 兼容格式）
    ├── System Prompt: JSON 输出格式 + 强制关键词
    ├── User Message: Profile + Position + JD
    └── 30 秒超时保护
    ↓
结果解析 → 打招呼文案 + 深度分析卡片
    ↓
自动创建投递记录
```

| 401 Unauthorized | "API Key 无效，请检查设置" |
| 429/402 余额不足 | "API 余额不足，请充值" |
