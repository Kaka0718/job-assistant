## 二、测试范围

### D1：拖拽排序（前端）
- 看板卡片能否垂直拖拽
- 跨列拖拽是否触发状态更新
- 拖拽失败时是否回滚
- 拖拽动画是否流畅

### D2：数据可视化（前端）
- 3 种图表是否渲染正常
- 图表数据与统计卡片是否一致
- 无数据时是否显示 EmptyState
- 图表响应式是否正常

### D3：导入/导出（前端）
- 导出 JSON 是否包含所有字段
- 导入 JSON 是否成功创建岗位档案
- 格式校验是否生效
- 字段校验是否生效

### D4：备份/恢复（前后端）
- 导出 zip 是否包含所有数据文件
- 从 zip 恢复后数据是否完整
- 恢复前 ConfirmDialog 是否弹出
- 恢复失败时数据是否回滚
